# Smart City Traffic Forecasting using Data Science and Machine Learning

## Project Description

This project predicts traffic flow using historical traffic data and Machine Learning techniques for smart city traffic management.

## Objectives

- Forecast traffic volume
- Analyze traffic patterns
- Support traffic management

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

## Machine Learning Algorithm

Random Forest Regressor

## Dataset

The dataset used for this project was provided as part of the training program and is not included in this repository.
