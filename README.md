# Upskill Campus Projects

This repository contains two Data Science and Machine Learning projects completed as part of the Upskill Campus training program.

## Projects

### 1. Agricultural Production Prediction
Predicts agricultural production using historical crop data and Machine Learning.

### 2. Smart City Traffic Forecasting
Predicts traffic flow using historical traffic data for smart city planning.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Jupyter Notebook
