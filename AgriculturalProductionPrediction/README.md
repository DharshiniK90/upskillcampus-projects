# Agricultural Production Prediction using Data Science and Machine Learning

## Project Description

Agriculture plays a vital role in food production and economic development. This project predicts agricultural production using historical crop data and machine learning algorithms.

## Objectives

- Predict crop production accurately
- Improve agricultural decision making
- Analyze historical crop data

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

## Machine Learning Algorithm

Random Forest Regressor

## Dataset

The dataset used for this project was provided as part of the training program and is not included in this repository.
